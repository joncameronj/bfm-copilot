## SECTION 2: AUTONOMIC PATTERN DECISION TREE

The HRV autonomic plot has FOUR distinct patterns. Each requires a DIFFERENT protocol.
Getting this wrong is the #1 source of incorrect recommendations.

### Pattern A: SNS SWITCHED (Upper-Left Quadrant)
- **What it looks like:** Red dot (stressed) clearly crosses to the SNS side (upper portion),
  blue dot stays on PNS side. SNS is clearly elevated and flipped.
- **Protocol:** SNS Balance (35 at 709)
- **Common mistake:** Do NOT trigger SNS Balance for any other pattern.

### Pattern B: LOWER-LEFT QUADRANT (Energetic Imbalance)
- **What it looks like:** Both SNS and PNS are negative/depleted — dots are in the lower-left
  quadrant. This is total energy depletion, NOT a sympathetic switch.
- **Protocol:** MIDBRAIN SUPPORT — this is energy restoration, not autonomic switching.
- **Common mistake:** DO NOT trigger SNS Balance for lower-left quadrant. The system is depleted,
  not switched. SNS Balance is wrong here.

### Pattern C: RED DOT SUPERIMPOSED ON BLUE DOT (Fight/Flight Freeze)
- **What it looks like:** The red (stressed) dot is directly on top of the blue (calm) dot.
  The nervous system cannot distinguish between calm and stressed states.
- **Protocol:** LOCUS COERULEUS frequency IN ADDITION to SNS Balance
- **Common mistake:** Missing the Locus Coeruleus — this pattern requires BOTH protocols.

### Pattern D: PNS NEGATIVE ZONE ONLY (Parasympathetic Failure)
- **What it looks like:** PNS is in the negative zone (below zero) but SNS is not clearly switched.
  This is vagal dysfunction without sympathetic switching.
- **Protocol:** PNS Support + Vagus Support (not SNS Balance)
- **Common mistake:** Triggering SNS Balance when only PNS is negative and SNS is NOT switched.

### HIGH BARORECEPTOR SENSITIVITY
- **Trigger:** HRV report explicitly notes "high baroreceptor sensitivity"
- **Protocol:** CSF SUPPORT frequency
- **Important:** Baroreceptor sensitivity must be at a **HIGH** level to trigger CSF Support.
  "Average" baroreceptor sensitivity does NOT trigger CSF Support. Only "high" does.
- **Common mistake:** Triggering CSF Support when baroreceptor sensitivity is average/normal.
  Also missed entirely sometimes — it's noted in the HRV interpretation text, not in the numeric data.

## SECTION 3: VAGUS NERVE PROTOCOL ORDER (Non-Negotiable)

The vagus protocols MUST be applied in this specific order:

1. **Vagus Support** — ALWAYS start here. This is the basic vagus support protocol.
2. **Vagus Balance** — Add ONLY if SNS is also switched (it addresses both vagus + SNS).
3. **Vagus Trauma** — Add ONLY if Vagus Support is insufficient OR the pattern specifically
   shows paradoxical deep-breathing response (deep breathing worsens PNS instead of helping).

**NEVER start with Vagus Trauma.** It is an escalation, not a first-line protocol.
**NEVER reverse the order** (Vagus Trauma before Vagus Support is always wrong).

## SECTION 4: CELL SYNERGY DOSING (Most Common Supplement Error)

Cell Synergy dosing rules — get this exactly right:

- **Standard dose:** 1 scoop/day. This is the default for ALL patients.
- **Double dose:** 2 scoops/day (NOT 4 scoops). Only increase to 2 scoops if:
  - "BE MORE ATTENTIVE" finding is present, OR
  - Alpha/Theta ratio is extremely uneven (>2:1) and not resolving after initial visits
- **NEVER recommend 3 or 4 scoops.** Double means 2, not 4.
- **Common mistake:** Recommending 2-3 scoops or 4 scoops as "double dose."
  Double = 2 scoops. Period.

## SECTION 5: SUPPLEMENT PHASING — DAY 1 vs WEEK 1-2 (Critical Timing)

This is the second most common error. Supplements are strictly phased:

### Day 1 (In-Office Testing Findings ONLY)
These supplements are triggered by in-office diagnostic findings. Recommend on Day 1:
- **Cell Synergy** — 1 scoop standard (all patients with pH issues or general depletion)
- **Tri-Salts** — ONLY if pH < 6.2 (not for pH between 6.2-6.5)
- **Pectasol-C** — ONLY if VCS failed (<24/32)
- **Serculate** — ONLY if Heart low on D-Pulse (<40%)
- **CoQ10** — ONLY if Heart low on D-Pulse (<40%)
- **X-39** — ONLY if protein positive in urine OR most/all D-Pulse organs are in single digits (stem cell support + UB rate). NOT automatic for every patient.
- **Innovita Vagus Nerve** — ONLY if vagus deal breaker triggered
- **D-Ribose** — ONLY if Theta >= 2.5x Alpha (e.g., Theta 25% / Alpha 10% = 2.5:1). NOT triggered by mild Theta > Alpha imbalance.
- **Deuterium Drops** — ONLY if specific gravity ≤ 1.005 on UA

### Week 1-2 (Lab Work Findings ONLY)
These supplements require lab confirmation. DO NOT recommend without lab results:
- **Vitamin D** — ONLY if Vitamin D lab < 60 ng/mL (target 60-80)
- **IP6 Gold** — ONLY if Ferritin elevated (MUST be empty stomach)
- **Homocysteine Factor** — ONLY if Homocysteine elevated
- **Adipothin** — ONLY if Leptin resistance confirmed (VCS fail + clinical picture)
- **Livergy** — ONLY if liver enzymes (AST/ALT) elevated
- **Pancreos** — ONLY if T2D (elevated glucose/HbA1c)
- **Kidney Clear** — ONLY if eGFR low / kidney function declining

### NEVER Recommend Without Labs
- **DO NOT** recommend Vitamin D if labs haven't been drawn yet
- **DO NOT** recommend Integra Cell if CRP hasn't been confirmed
- **DO NOT** assume labs will be abnormal — wait for actual results
- **Common mistake:** Recommending Week 1-2 supplements on Day 1 before labs are back.

## SECTION 6: X-39 TIMING (Conditional Rule)

**X-39 is NOT automatic for every patient.** It is a Day 1 recommendation ONLY when triggered by:
1. **Protein positive in urine** (protein spilling indicates stem cell / UB rate support needed)
2. **Most or all D-Pulse organs in single digits** (severe systemic depletion requiring stem cell stimulation)

If neither condition is met, do NOT recommend X-39.

When prescribed:
- Place at C7, below belly button, or over specific injury site
- 12 hours on / 12 hours off, rotate placement every 2-3 days
- **Common mistake:** Recommending X-39 universally for all patients. It requires a specific trigger.

## SECTION 7: VCS → LEPTIN → MSH CASCADE (Gating Rule)

When VCS fails, this cascade is activated. The sequence is NON-NEGOTIABLE:

1. **VCS Failed** → Biotoxins present
2. **Biotoxins** → Cause leptin resistance
3. **Leptin resistance** → Causes low MSH
4. **Low MSH** → Downstream hormone problems

### Treatment Order (MUST follow this sequence):
**FIRST:** Address Leptin
- Biotoxin frequency + Leptin Resist frequency
- Adipothin supplement + leptin-style diet education
- Pectasol-C (biotoxin binder)

**THEN (only after Leptin is trending down):** Address MSH
- Pit A Support frequency
- Pars Intermedia frequency
- Hypothala supplement (contains MSH codes)

**NEVER** jump to MSH protocols (Pit A Support, Pars Intermedia) before Leptin is addressed.
This is called the "Cyto Lower Gate" — Cyto Lower cannot run until MSH/Leptin Resist is resolved.

## SECTION 9: MONITORING & RETEST TIMING

### Lab Retesting (LabCorp/Bloodwork)
- **Retest interval:** 3 MONTHS (not 6 weeks)
- **Common mistake:** Recommending lab retest at 6 weeks. The correct interval is 3 months.

### In-Office Retesting (DePulse, HRV, Ortho, Valsalva, UA, VCS)
- **Retest interval:** Every 2-4 weeks
- These are non-invasive and should be repeated regularly to track progress

## SECTION 10: SESSION FREQUENCY CONTEXT

- **Standard BFM practice:** Patients are seen 2× per week. This is the baseline.
- **"BE MORE ATTENTIVE" finding:** Means MORE than standard — recommend 3×/week or more.
- **DO NOT** frame standard 2×/week as exceptional. It is normal BFM standard of care.
- **DO NOT** suggest patients need external conventional medical co-management in the report.
  This is a BFM clinical document — keep recommendations within BFM framework.
